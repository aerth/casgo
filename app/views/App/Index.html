Static Contact

The easiest way to add a contact form to any website -

have the form sent directly to your inbox.

Copying and pasting an email address or launching a program can interrupt a user's experience.

An inline form maintains user focus and increases engagement.

Quick to set up

Simply change the action attribute of a form.

Perfect for any page

Static Contact is the easiest and fastest way to set up a contact form on any site, regardless of hosting provider or tech stack.

Absolutely free

Yep, free. Also, no messages are or ever will be stored.

All that is required is to set the action attribute on a HTML form to be "http://www.staticcontact.com/contact/{your_email}"

The name, email, and message elements are required and must have those names. The subject element is optional.

Typically, on submission of a form, a user is redirected to an alternate page. The javascript below (which requires jQuery) will submit the form but not redirect the user.
